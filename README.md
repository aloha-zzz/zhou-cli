# my-Cli

目前仅支持React 和 Koa2 模板构建
1. React 
    - ES6
    - webpack < 4
      - development mode (webpack-dev-server)
      - production mode  
    - ESLint
    - React-Router4
2. Koa2 (node >= 7.6) 
    - mysql
    - gzip
    - static resource
    - Error Catch
    - wrongLogs
    - logger
    - deamon
    - CORS
    - cookie/session
